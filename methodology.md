# Methodology

## Approach

This project uses prompt-based essay generation to exploit differences in LLM evaluation.

Three main writing strategies are used:

1. Philosophical
   - Abstract reasoning
   - Ethical ambiguity
   - Contradictory ideas

2. Metaphorical
   - Rich imagery
   - Layered meaning

3. Dialectical
   - Competing viewpoints
   - Logical tension

---

## Model

A transformer-based language model is used to generate essays.

---

## Evaluation

Multiple LLM judges are simulated using random scoring.

Metrics used:
- Average Quality Score
- Standard Deviation (Disagreement)
- Engagement Score

---

## Goal

Maximize disagreement score between judges rather than correctness.
